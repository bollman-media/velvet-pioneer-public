Particles spiral toward the centroid of painted erase points — reinforcing the "sucking in" erase metaphor. Dashed circle = vortex target.
Vortex centroid calculation
const sumX = erasePoints.reduce((s,p) => s+p.x, 0); const sumY = erasePoints.reduce((s,p) => s+p.y, 0); vortexCenter = erasePoints.length > 0 ? { x: sumX / erasePoints.length, y: sumY / erasePoints.length } : null; // null → falls back to image center
